# 썸네일 뱃지(배너) 이미지 자리

여기에 썸네일에 얹을 **뱃지 PNG**를 둡니다.

## 현재 뱃지 (위→아래 표시 순서)

| 파일명 | 내용 | 크기 |
|---|---|---|
| `01_badge.png` | 追跡付 (추적첨부가능) | 191×99px |
| `02_badge.png` | 最安値挑戦 (최저가 도전) | 280×99px |
| `03_badge.png` | 送料・関税込 (우송료·관세 포함) | 305×99px |

- 좌측 하단에 위 순서대로(01 맨 위) 세로로 쌓입니다.
- 파일명·순서를 바꾸려면 `../thumbnail_generator.py` 상단의 `BADGES` 목록을 수정하세요.
- **그림자 없는 투명 PNG**를 넣습니다. 그림자(우하단 x10 y10 blur5 검정)는
  코드가 스펙대로 자동으로 만들어 얹습니다.
  - 이미 그림자가 그려진 PNG를 쓰려면 `SHADOW_ENABLED = False` 로 두세요.

## 현재 설정값 (`thumbnail_generator.py`)
- `BADGE_SCALE = 1.15` — 뱃지 확대 배율 (원본 191/280/305 → 약 220/322/351)
- `SHADOW_OPACITY = 0.65` — 그림자 진하기 65%
- 여백 좌·하단 30px, 뱃지 간 간격 20px, 세로 스택(`ARRANGEMENT="vertical"`)
- 가로로 나열하려면 `ARRANGEMENT = "horizontal"`
