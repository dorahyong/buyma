## 3. 옵션 파싱 상세

`option_stock_data` 원본 (단일 JSON 문자열로 이스케이프됨):
```json
{
  "P000BBSY000C": {
    "option_value": "M",
    "stock_number": 2,
    "is_selling": "T",
    "is_display": "T",
    "use_soldout": "T",
    "option_price": 752000,
    ...
  },
  ...
}
```

추출 로직 (9tems와 동일):
```python
m = re.search(r"var\s+option_stock_data\s*=\s*'(\{.*?\})'\s*;", html)
if m:
    stock_data = json.loads(m.group(1))
    for opt_code, opt in stock_data.items():
        opt_value = opt.get('option_value', '')
        stock_number = int(opt.get('stock_number', 0) or 0)
        is_selling = opt.get('is_selling', 'T') == 'T'
        status = 'in_stock' if (is_selling and stock_number > 0) else 'out_of_stock'
        options.append({
            'color': '',
            'tag_size': opt_value,
            'option_code': opt_code,
            'status': status,
        })
```

images = []
_seen = set()
big = soup.select_one('div.detailArea img.BigImage')
if big and big.get('src'):

    if src:

model_imgs = []
for img in soup.select('div#prdDetail img[src]'):

    if 'loromoda123.cafe24.com' in src and SABANG in src and model_id.upper() in src.upper():

        model_imgs.append(url)
# 마지막 1장 제외 (사용자 메모)
if len(model_imgs) >= 2:

for url in model_imgs:
