### Values

순서대로 나열해주시면 됩니다.

1. 이미지 : 일단 표 자체에서도 대표이미지 1장은 보입니다.  클릭하면 팝업을 띄우고 아래의 내용을 보여줍니다. 표 형태로 만들어주세요.
    1. 수집된 모든 이미지들
    2. 상품 업로드시 활용될지 여부. “사용”, “미사용” 으로 분류
    3. 각 이미지의 출처명
    4. 각 이미지의 출처 상품 링크 
2. 상태 : 아래와 같은 상품상태를 갖습니다.
    1. 출품중 : 바이마에 출품중입니다.
    2. 출품대기중 : 5만개 자리가 꽉차서 못올린거지, 올릴 수 있는 상태의 상품입니다.
    3. 품절 : 소싱처 품절로 못올리게된 상품입니다.
    4. 최저가확보불가 : 소싱처에 재고는 있으나 최저가 확보가 안된 상품입니다.
    5. ~~출품보류중 : 최저가도 확보되고 품절도 아니나 매력도가 떨어져, 혹은 그 외 사유로 출품을 안하게된 경우입니다. 이런경우가 아직은 없을텐데 그래도 상태정도는 만들어두겠습니다.~~ 
3. 상품명(일본어) : 우리가 등록한, 혹은 등록할 상품명입니다.
4. 상품명(한국어) : 등록한, 혹은 등록할 상품명의 한글번역입니다.
5. 총 조회수 : 해당 상품의 총 액세스수 입니다.
6. 총 장바구니수 : 해당 상품의 총 장바구니수 입니다.
7. 총 찜 수 : 해당 상품의 총 찜 수 입니다.
8. 일평균 찜수 : 총찜수/바이마등록기간
9. 일평균 조회수 : 총조회수/바이마등록기간
10. 일평균 장바구니수 : 총장바구니수/바이마등록기간
11. 바이마 최저가 : 동일품번의 바이마 최저가입니다.
12. 출품가능 최저가 : DB상 우리가 출품 가능한 최저가 입니다.
13. 바이마 출품가 : 우리가 바이마에 출품한 출품가 입니다. 출품중이 아닌 경우 0으로 표시합니다.
14. 기대마진 : konnect 2026 파일의 계산식을 적용하여 기대마진을 계산해서 넣습니다. 마이너스나서 못파는 상품은 마이너스값으로 표시합니다. 자세한 계산식 관련해서 궁금한점이 있으시면 코넥트 단톡방에 말씀해주세요.
15. 인기순 순위 : 바이마 내 해당 품번의 인기순 상 우리의 순위가 몇위인지를 표시합니다. 품번이 여러개일 경우 평균값으로 표시합니다.
    1. 클릭시 팝업으로 각 품번별 순위를 보여줍니다.
16. 동일품번 제품수 : 바이마 내 동일품번의 제품수를 보여줍니다. 품번이 여러개일 경우 여러 품번의 조회값 전체에서 중복을 제외한 후 제품수를 내서 알려줍니다.
17. 동일품번 총조회수 : 바이마 내 동일품번의 제품수와 동일하게 처리 후 각 상품들의 액세스수를 합쳐서 보여줍니다.
18. 동일품번 인기 1위상품 : 품번별 인기순 1위 상품을 보여줍니다. 클릭시 팝업을 띄워주고 팝업에는 아래의 내용이 들어갑니다. 
값 명은 타제품이 1위인 경우 “확인하기” 라고 표시하고, 만약 우리제품이 인기순 1위일 경우 “우리가 차지함” 이라고 표시합니다.  
    1. 링크
    2. 조회수
    3. 문의수
    4. 최근 문의일
19. 최근 7일 조회수 : 상품 편집 페이지에 보면 상단에 표시되어있습니다. 그 값을 가져와서 보여주면 됩니다.
20. 바이마 상품링크 : 바이마 해당 상품으로 가는 링크입니다.
21. 상품출처 :클릭하면 팝업을 띄우고 링크들을 보여줌. 중복된 곳들 있으면 나열해줌
22. 최저가 업데이트 일시 : 바이마에 우리가 최저가업데이트를 한 최근일시 입니다. 소싱처 업데이트하면서 최저가 업데이트해도 갱신되고, 최저가만 관리하는 프로그램이 돌다가 업데이트를 해도 갱신됩니다.
23. 소싱처 업데이트 일시 : 소싱처 업데이트 일시 입니다.
24. 바이마 등록일 : 바이마에 상품을 등록한, 해당 상품의 등록일입니다.
25. 바이마 등록기간 : 해당 상품등록일로부터 오늘까지 며칠지났는지 입니다.
26. 바이마 구매기한 : 바이마에 상품 등록하면 나오는 해당 상품의 구매기한 입니다.
